## 📝 **إنشاء حساب طبيب**

### المستندات المطلوبة
| المستند | الوصف |
|---------|-------|
| 🆔 بطاقة الهوية | صورة بطاقة الهوية أو جواز السفر |
| 📜 رخصة مزاولة المهنة | صورة الترخيص الطبي |

### البيانات المطلوبة
الاسم ← التخصص ← المستشفى ← المدينة ← المنطقة ← الرخصة ← الجوال ← البريد ← كلمة المرور ← نبذة تعريفية

> 📌 يتم توثيق الحساب بعد مراجعة المستندات من قبل المشرفين

## 👥 **إدارة المرضى**

### عرض المرضى
- 🔍 **بحث** بالاسم
- 📇 **بطاقة المريض:** الاسم، العمر، الجنس، التاريخ الطبي، الملاحظات

### إضافة مريض جديد
**"إضافة مريض"** ← الاسم ← العمر ← الجنس ← الجوال ← التاريخ الطبي ← الملاحظات ← **"حفظ"**

### تفاصيل المريض
- 📞 معلومات التواصل
- 📋 التاريخ الطبي والملاحظات
- 💊 **الوصفات الطبية:**
  - اسم الدواء، الجرعة، التكرار، المدة
  - ➕ إضافة وصفة / 🗑️ حذف وصفة
- 📅 المواعيد السابقة

---

## 💳 **الإدارة المالية**

### نظرة عامة
- 💰 إجمالي الإيرادات
- 💳 المدفوعات الإلكترونية
- 📄 عدد الفواتير

### تسجيل معاملة مالية
**"تسجيل معاملة مالية"** ← اسم المريض ← المبلغ ← طريقة الدفع ← الوصف ← **"حفظ"**

> ✅ يتم تسجيل المعاملة تلقائياً عند إنهاء موعد

## ⭐ **الباقة والاشتراك**

### خطط الاشتراك
| الباقة | شهرياً | الحد الأقصى |
|--------|--------|-------------|
| 🆓 مجانية | 0 ج.م | 10 مرضى |
| 🔵 أساسي | 99 ج.م | 50 مريضاً |
| 🟣 مميز | 199 ج.م | غير محدود |

### 🎁 فترة التجربة
- **3 أيام** باقة مميزة مجاناً بعد التسجيل
- ⏳ أيام متبقية تظهر في لوحة التحكم
- 🔄 تحويل تلقائي للمجانية بعد انتهاء التجربة

### خيارات الدفع
- 📆 شهرياً
- 📆 سنوياً (مع توفير)

## 👤 **الملف الشخصي للطبيب**

### البيانات القابلة للتعديل
- 📝 الاسم، رقم الجوال، المدينة، المنطقة، الرخصة، النبذة

### المستندات
- 🆔 بطاقة الهوية — عرض / رفع / استبدال
- 📜 رخصة مزاولة المهنة — عرض / رفع / استبدال

### شركات التأمين
- ➕ إضافة شركات تأمين
- ➖ إزالة شركات تأمين

### حالة التوثيق
- ✅ **موثّق** — تم التحقق من المستندات
- ⏳ **قيد الانتظار** — لم يتم التحقق بعد
