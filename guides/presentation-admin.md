## 👨‍⚕️ **إدارة الأطباء**

![bg left:30% 85%](https://placehold.co/300x500/8B0000/white?text=Doctors)

### قائمة الأطباء
الاسم ← التخصص ← المستشفى ← المدينة ← البريد ← الرسوم ← التوثيق ← الرخصة

### الإجراءات
| الإجراء | الوصف |
|---------|-------|
| 🔍 عرض التفاصيل | معلومات كاملة + مستندات + اشتراك |
| ✏️ تعديل الطبيب | تعديل كل البيانات والتوثيق |
| ⭐ تعيين اشتراك | تغيير خطة الاشتراك |
| 🔑 تغيير كلمة المرور | إعادة تعيين |
| 🗑️ حذف الطبيب | إزالة من النظام |

## 📥 **استيراد وتصدير الأطباء**

### استيراد من CSV
- 📂 رفع ملف CSV
- 🌐 يدعم الرؤوس بالعربية والإنجليزية
- ⚡ استيراد دفعة واحدة

### تصدير إلى CSV
- 📤 تصدير جميع الأطباء
- 🔤 ترميز UTF-8 مع BOM (متوافق مع Excel)

## 👥 **إدارة المرضى**

### قائمة المرضى
| العمود | الوصف |
|--------|-------|
| الاسم | اسم المريض |
| البريد الإلكتروني | للتواصل |
| الجوال | رقم الهاتف |
| تاريخ التسجيل | تاريخ الانضمام |

### تفاصيل المريض
- 📞 معلومات التواصل (الاسم، البريد، الجوال، تاريخ الميلاد، الجنس)
- 💬 المحادثات (قابلة للضغط)
- 📅 المواعيد (مع الطبيب والحالة)
- 🔑 تغيير كلمة المرور

---

## 📅 **إدارة المواعيد**

### فلترة حسب الحالة
🔄 الكل ← ⏳ معلّق ← ✅ مؤكد ← ✔️ مكتمل ← ❌ مرفوض

### قائمة المواعيد
المريض → ← الطبيب ← التخصص ← المستشفى ← الموعد ← الجوال ← الحالة

### الإجراءات
| الإجراء | الحالة المطلوبة |
|---------|-----------------|
| ✅ قبول | معلّق |
| ❌ رفض | معلّق |
| ✔️ إنهاء | مؤكد |
| ✏️ تعديل | أي حالة |

---

## ⭐ **إدارة الاشتراكات**

### جدول الاشتراكات
| العمود | الوصف |
|--------|-------|
| الطبيب | اسم الطبيب |
| الباقة | مجاني / أساسي / مميز |
| تاريخ البدء | بداية الاشتراك |
| تاريخ الانتهاء | نهاية الاشتراك |
| تجربة | نعم / لا |
| الحالة | نشط / منتهي |

### الإجراءات
- تعيين خطة جديدة
- تحديد تاريخ انتهاء مخصص

---

## 🔐 **إدارة المشرفين**

### قائمة المشرفين
🆔 المعرف ← 👤 الاسم ← 📧 البريد ← 🔑 الصلاحية ← 📅 تاريخ الإنشاء

### الإجراءات
| الإجراء | الوصف |
|---------|-------|
| ➕ إضافة مشرف | الاسم + البريد + كلمة المرور |
| 🔑 تغيير كلمة المرور | لأي مشرف |
| 🗑️ حذف مشرف | ❌ لا يمكن حذف نفسك |

### الصلاحيات
- 🔰 **مشرف عام:** كافة الصلاحيات
- 👤 **مشرف:** صلاحيات محدودة

## ❓ **أسئلة شائعة**

| السؤال | الإجابة |
|--------|---------|
| كيف أوثق طبيباً؟ | "الأطباء" ← "تعديل الطبيب" ← تفعيل "موثّق" |
| كيف أستورد أطباء؟ | "الأطباء" ← "استيراد أطباء" ← CSV |
| كيف أعيد تعيين كلمة المرور؟ | اختر المستخدم ← "تغيير كلمة المرور" |
| هل يمكن حذف مشرف؟ | ✅ نعم، عدا نفسك |
| كيف أغير خطة اشتراك؟ | "الأطباء" ← "تعيين اشتراك" |
